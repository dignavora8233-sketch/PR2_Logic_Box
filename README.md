# 🚀 Logic Box

**A beginner-friendly Python project that generates star patterns and analyzes a range of numbers through an interactive command-line application.**

# 📖 Introduction

**Logic Box** is a simple Python project created to strengthen programming fundamentals. It provides a menu-driven interface where users can generate star patterns or analyze a range of numbers by identifying even and odd numbers and calculating their total sum.

This project is ideal for beginners who want to practice loops, conditional statements, user input, and basic arithmetic operations in Python.

# ✨ Features

- ⭐ Interactive menu-driven program
- 🔹 Generate a star (`*`) pattern
- 🔢 Analyze numbers in a given range
- ✅ Identify Even and Odd numbers
- ➕ Calculate the total sum of numbers
- 🔁 Continuous execution until the user chooses to exit

# 🛠️ Technologies Used

- Python 3
- Command Line Interface (CLI)
